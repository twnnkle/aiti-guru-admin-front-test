### Информация о использовании ИИ:
ИИшка использовалась в основном для поиска информации, а не для генерации кода. Местами использовался автокомплит от курсора. Для генерации кода использовал обычный чат Deepseek для следующих вещей (для каждого пункта была сгенерирована только база для скорости, остальное допиливал сам):
1. На форме авторизации описание с градиентом
2. Для роутера
3. База для таблицы
4. База для модалки
5. Для масок

Старался не использовать особо автокомплит и генерацию кода, чтобы показать свои способности. Если бы использовал на полную катушку, то было бы еще в разы быстрее
